# SNN Stuff

This directory holds projects dedicated to **spiking graph neural networks** (SGNNs).

Spiking graph neural networks combine:
- **Spiking neural networks (SNNs)** — event-driven, bio-inspired models that use discrete spikes instead of continuous activations.
- **Graph neural networks (GNNs)** — models that operate on graph-structured data (nodes, edges) for relational and structural reasoning.

Here you will find implementations, experiments, and related code for SGNN research and development.
